# Description
ggplot2を使った作図。    
RStudioで管理している。

## Usage
* プロットした図は、`figure`に保存すること
* プロットに用いる計算結果のデータは、`result_data`に保存すること
* 1ファイルで1プロットを原則とし、Rscriptの使い回しやレジェンドの変更などはしない。その場合は別のスクリプトとして保存する。
